# Python ATM Simulator

## Overview

This project is a simple ATM Simulator built using Python. It allows users to perform basic banking operations such as checking account balance, depositing money, withdrawing money, and exiting the application.

The project was created to practice Python fundamentals, conditional statements, loops, and user interaction.

---

## Features

* Check account balance
* Deposit money
* Withdraw money
* Prevent withdrawal when balance is insufficient
* Exit the application safely
* Menu-driven interface

---

## Technologies Used

* Python

---

## Concepts Used

* Variables
* User Input
* While Loops
* If-Else Statements
* Nested Conditions
* Arithmetic Operations

---

## How It Works

1. The program starts with an initial account balance.
2. Users can select options from the ATM menu.
3. Users can:

   * Check Balance
   * Deposit Money
   * Withdraw Money
   * Exit
4. The balance is updated after each transaction.
5. The program prevents withdrawals that exceed the available balance.
